This repository documents hands-on Identity & Access Management (IAM) labs built using Microsoft Entra ID (Azure AD).
The labs demonstrate foundational IAM capabilities including identity lifecycle management, RBAC, MFA enforcement, enterprise application access control, and audit monitoring.

> Note: This repository contains sanitized documentation only.
> Detailed evidence and screenshots are maintained privately.

## Technologies Used
- Microsoft Entra ID (Azure AD)
- Azure Free Account
- Security Defaults
- Role-Based Access Control (RBAC)
- Enterprise Applications
- Identity Lifecycle Management (JML)

## Lab Overview
- Lab 0: Tenant, Users, and Groups
- Lab 1: RBAC and Least Privilege
- Lab 2: MFA and Security Defaults
- Lab 4: Enterprise Application Access Control
- Lab 5: Joiner / Mover / Leaver (Identity Lifecycle)
- Lab 6: Monitoring, Sign-In Logs, and Audit Evidence
