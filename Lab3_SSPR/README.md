# Lab 3 — Self-Service Password Reset (SSPR)

Configured SSPR for Boclair Industries to allow users to reset their own 
passwords without contacting the help desk.

## What was configured
- SSPR enabled for a pilot group (SSPR-Pilot-Users)
- Authentication methods: Email + Mobile app code
- Registration required on next sign-in
- Notifications: Users notified on reset, admins notified when 
  other admins reset
