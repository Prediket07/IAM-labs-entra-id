# Lab 2 — MFA & Authentication Methods

Configured multifactor authentication methods and enabled the MFA 
registration policy in Microsoft Entra ID Identity Protection.

## What was configured
- Microsoft Authenticator: Enabled for all users
- Number matching: Enabled (MFA fatigue attack defense)
- Authentication mode: Any (passwordless + push notification)
- MFA registration policy: Enabled for all users — forces 
  registration at next sign-in
